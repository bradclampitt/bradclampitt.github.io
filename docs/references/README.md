## Mobile App - Required Connections
These extensions or services will need to be interconnected from the Mobile App to our website through API or some other fashion.
**Note:** There may be more of our extensions that need to be directly tied in, some that are through Checkout and are blended into the checkout process I did not add below but they may need to be reviewed, things like ShipperHQ, AvaTax, Multi Fees, etc.

- Rootways - Authorize.net CIM extension
    >   For Credit Cart Payments, Apple Pay and in the future Google Pay
    >   Set Payment Method, Update Method, Save and Delete Cards, etc.
- Aheadworks - Blog

Functionality to replicate in the mobile apps that are apart of our desktop site
- Customer Profile Editing (Billing & Shipping Addresses, Account Details, etc)
- Customer Orders (Order Tracking and Updates, View Orders, etc)
- Product Categories - Filtering Options that mimics the website (filtering to get an overhual soon)
- Product Details (Custom Descriptions, Custom Configuration Options, Tags/Attributes)
- ShipperHQ Shipping Quotes (Includes; No Ship States/Countries and any other restrictions)
- AvaTax Taxation with our Multi-Fees 1% Handling Costs
- Gift Cards (Redeem, add messaging, purchase cards, gift cards)
- Wine Requests (Create Requests, List Wines, Update, Upload Images, Get Messages, Set Notifications)
- CMS / Static Content
- CMS Banners, Product Carousels
